# SENTI/VAL v2.0

**23rdfloorinc.+FROGULA+LABS+AI PRESENTS: SENTI/VAL**

A visually stunning OSINT dashboard with an interactive WebGL particle orb that reacts to scan operations.

## 🌟 Features

- **100,000 Particle WebGL Orb** - 5 morphological states (Astrolabe, DNA Helix, Core Pulse, Neural Net, Data Stream)
- **Reactive Visualizations** - Orb changes shape and pulses on each scan
- **8 OSINT Domains** - Identity, Infrastructure, Social, Financial, Geospatial, Darkweb, Corporate, Media
- **Cyberpunk UI** - Glass morphism, neon accents, grid overlays
- **Real-time Intelligence** - IP geolocation, DNS lookup, GitHub profiling, crypto wallet tracing

## 🎮 Usage

1. Enter a target (domain, IP, @username, wallet address)
2. Select an OSINT domain
3. Click **INITIATE SCAN**
4. Watch the orb react to new data nodes!

## 🎨 Tech Stack

- HTML5 / CSS3 / JavaScript (ES6+)
- Three.js (WebGL particle system)
- Tailwind CSS (utility classes)
- Custom GLSL Shaders
